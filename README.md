# restart-assingment

১) Null vs Undefined

Null: আমরা নিজে ভ্যারিয়েবলকে খালি বানাই।

Undefined: ভ্যারিয়েবল বানিয়েছি কিন্তু কোন মান দিইনি, তাই ব্রাউজার বলে “undefined”।

২) map() vs forEach()

map(): কোনো অ্যারেকে নিয়ে নতুন অ্যারে বানায়।

forEach(): শুধু অ্যারের সব আইটেমের জন্য কাজ করে, নতুন কিছু বানায় না।

৩) == vs ===

==: মান মিলিয়ে দেখে, টাইপ ভিন্ন হলেও ঠিক করতে চেষ্টা করে।

===: মান আর টাইপ দুটোই একসাথে মিলতে হবে।

৪) Async/Await কেন দরকার?

API থেকে ডেটা আনা সহজ হয়।

কোড পড়তেও পরিষ্কার হয়।

৫) Scope (Global, Function, Block)

Global: পুরো প্রোগ্রামে ব্যবহার করতে পারি।

Function: ফাংশনের ভিতরে ব্যবহার করতে পারি।

Block: {} ব্লকের ভিতরে ব্যবহার করা যায়, সাধারণত let বা const দিয়ে।
